How to Use Anyshift with PCSX2

Full setup guide for playing PS2 racing games with a force feedback wheel and H-pattern shifter using PCSX2 1.6.0.

📋 On this page

  1. Requirements
  2. USB Wheel Plugin Setup
  3. Wheel Axis & Button Config
  4. Gamepad / Controller Config
  5. Configure Anyshift
  6. Supported PS2 Games

Requirements

  • PCSX2 1.6.0 — newer versions do not support force feedback wheels at the time of writing. Use exactly version 1.6.0 for FF support.
    Download PCSX2 1.6.0 →
  • A force feedback steering wheel with DirectInput support
  • An H-pattern gear shifter recognised by Windows
  • Anyshift installed and working
  • Your PS2 game ISO (Gran Turismo 4, Enthusia, Tokyo Xtreme Racer 3, etc.)
⚠ PCSX2 Version
If you don't need force feedback, newer versions of PCSX2 (PCSX2-Qt) may work too. But for full force feedback support on Logitech/Thrustmaster/Fanatec wheels, stick with PCSX2 1.6.0.

Step 1 — USB Wheel Plugin Setup

PCSX2 emulates a PlayStation 2 USB wheel via its USB plugin. You need to tell it which wheel model to emulate — different games work best with different wheel types.

  1. Open USB Plugin Settings

    In PCSX2 1.6.0, go to Config → USB → Plugin Settings.

  2. Set Device type to "Wheel device"

    Under Device type — Port 1, select Wheel device. Leave Port 2 as None (unless you need a second controller).

  3. Set Device API to "DInput"

    Under Device API — Port 1, select DInput. This uses the DirectInput driver, which is required for force feedback.

  4. Select the emulated wheel model

    Under Emulated wheel — Port 1, choose the wheel model that matches what the game expects. See the table below.

    GameEmulated wheel
    Gran Turismo 4 Driving Force Pro (rev11.02)
    Enthusia Professional Racing Driving Force Pro
    Tokyo Xtreme Racer 3 Driving Force
    Tokyo Xtreme Racer Drift 2 Driving Force

  5. Click OK to save

Step 2 — Configure Wheel Axes & Buttons

After saving the USB plugin settings, click Configure next to Port 1 to open the wheel axis/button configuration window.

Mapping your wheel

The configuration window shows all available inputs. Map these at a minimum:

Plugin input Map to Notes
Steer Left / Steer Right Steering wheel axis (left/right) Use the full steering axis of your wheel
Throttle Accelerator pedal axis Separate pedal axis if your wheel supports it
Brake Brake pedal axis Separate pedal axis if your wheel supports it
ℹ Pedals — separated vs combined
For GT4 and Enthusia, separate pedal axes are supported and recommended. The wheel configuration should have individual throttle and brake axes. Enable independent pedals in your wheel software if available.

Force Feedback settings

In the right panel of the wheel config window:

  • Enable Force Feedback for the best experience.
  • Adjust Gain to taste (start at 75%).
  • Enable Bypass Startup Calibration to skip the startup wheel spin.

Shift buttons — Tokyo Xtreme Racer only

Tokyo Xtreme Racer 3 and Drift 2 use L1 / R1 for shifting and read input from the USB wheel plugin — not the gamepad plugin. For these games, map the shift buttons directly in this wheel config window:

Game actionMap to keyAnyshift
L1 (downshift) Z Downshift key
R1 (upshift) S Upshift key
Circle (reverse) C Reverse key
⚠ Note for TXR3 / Drift 2
When you bind shift keys in the USB wheel plugin, the wheel paddles lose their original function. If you want to use wheel paddles for shifting, rebind them here.

Step 3 — Gamepad Controller Config (GT4 & Enthusia)

For Gran Turismo 4 and Enthusia, the shift and reverse buttons are read through the gamepad (PAD) plugin, not the USB wheel plugin. This approach lets you keep using the wheel paddles for shifting if you want — the gamepad binding coexists with the paddle binding.

  1. Open the Gamepad Plugin Settings

    Go to Config → Controllers (PAD) → Plugin Settings.

  2. Go to the Pad 1 tab

    Select the Pad 1 tab in the LilyPad settings window.

  3. Bind the shift keys using your keyboard

    Under the Device: WM Keyboard entries, assign the following keyboard keys to the corresponding PS2 gamepad buttons:

    Gran Turismo 4

    PS2 buttonGame functionKeyboard key
    L2 Downshift Z
    R2 Upshift S
    Triangle (△) Reverse C

    Enthusia Professional Racing

    PS2 buttonGame functionKeyboard key
    L2 Downshift Z
    R2 Upshift S
    — Reverse (it's a gear, not a button) —

  4. Click Aceptar / OK to save

✅ Wheel paddles still work
By binding shift keys in the PAD plugin (gamepad), your physical wheel paddles continue to work as upshift/downshift — they coexist with the Anyshift key simulation. This means you can shift with either the H-shifter or the paddles.

Step 4 — Configure Anyshift

  1. Launch Anyshift

    Run Anyshift.exe. Run as Administrator if it has trouble sending keystrokes.

  2. Load a preset

    In Profiles, pick the preset for your game and click Load profile. Memory write presets are also available.

    GameKey Press presetMemory Write preset
    Gran Turismo 4 Gran_Turismo4_pcsx2 Gran_Turismo4_pcsx2_memMode
    Enthusia Enthusia_pcsx2 Enthusia_pcsx2_memMode

  3. Select your shifter and configure buttons

    In Joystick selection, choose your H-shifter. Assign each gear in the Joystick Buttons section.

  4. Verify Key Selection

    Confirm the keys in Anyshift match what you set in the PCSX2 PAD plugin:

    Anyshift fieldDefault
    UpshiftS
    DownshiftZ
    ReverseC

  5. Click "Run Anyshift", then start PCSX2

    Click Run Anyshift. The window freezes — that's normal. Open PCSX2 and load your game. Select manual transmission in the game options before getting in the car.

    Press End to stop Anyshift.

Supported PS2 Games

Game details

  • Tested with: NTSC US version
  • Supports native 1080i and 60fps
  • Force feedback: ✅
  • 900° wheel rotation: ✅
  • Independent pedals: ✅
  • Memory Write mode: ✅ (reverse slightly buggy in mem mode)

On rolling starts, engage the correct starting gear before crossing the start line to avoid desynchronisation.

Engage 1st gear before you start accelerating — the game will engage 1st automatically when you accelerate. The game autoshifts to neutral when stopped, so re-select 1st gear each time you come to a stop.
