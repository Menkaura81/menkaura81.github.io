How to Use Anyshift with DOSBox

Use your H-pattern shifter with classic DOS racing games — NASCAR Racing, IndyCar Racing, Grand Prix 2, and more.

📋 On this page

  1. Requirements
  2. Wheel Setup
  3. Configure Anyshift
  4. Running
  5. Supported DOSBox Games

Requirements

  • DOSBox 0.74 — for Memory Write mode (Grand Prix 2) you must use exactly this version. The base memory address changes in 0.74-2 and 0.74-3. For key press mode (NASCAR, IndyCar…) any recent version works fine.
    Download DOSBox 0.74 from SourceForge →
  • A force feedback steering wheel (Logitech, Thrustmaster, Fanatec, etc.)
  • An H-pattern gear shifter recognised by Windows as a joystick
  • Anyshift installed and working

Step 1 — Configure Your Steering Wheel

Most DOS racing games only support one throttle/brake axis (not separate pedals). You need to set your wheel to combined pedal mode before launching the game.

  1. Open your wheel's configuration software

    For Logitech wheels: open Logitech Gaming Software or G Hub. For Thrustmaster: use Thrustmaster Control Panel. Navigate to the settings for your wheel.

  2. Enable combined pedals

    Look for a "Pedal detection mode" or similar option. Select Combined (single axis). This merges throttle and brake into one axis, which is what DOS games expect.

  3. Set steering rotation degrees

    DOS games were designed for wheels with limited rotation. Set to 270° (approximately the lock-to-lock range of wheels from that era). This gives you proportional steering without over-rotating.

    For IndyCar and NASCAR (oval racing) you may prefer 400° for more precise high-speed steering.

  4. Enable Force Feedback centering (optional)

    DOS games generally don't output force feedback effects, but enabling the wheel's self-centering spring gives a more realistic feel. Set centering intensity to taste (100% for a firm feel).

ℹ Note
The combined pedal setting must remain active for the entire session. Some wheel software resets it on reboot — check before each play session.

Step 2 — Configure Anyshift

  1. Launch Anyshift

    Run Anyshift.exe. If nothing happens or it errors, try running as Administrator.

  2. Load a preset (if available)

    In the Profiles section, open the dropdown and pick the preset for your game. Click Load profile.

    GamePreset file
    NASCAR RacingNascar_Racing_DOSBox
    NASCAR Racing 2Nascar_Racing2_DOSBox
    IndyCar RacingIndyCar_Racing_DOSBox
    Grand Prix 2Grand_Prix2_DOSBox_memMode

    If your game has no preset, continue with manual setup below.

  3. Select your shifter device

    In Joystick selection, choose your H-shifter from the dropdown. If you use a clutch, tick the Clutch checkbox and select the clutch device. Set the correct Clutch Axis number and Bitepoint.

  4. Assign shifter buttons

    Click each gear position button in the Joystick Buttons box, then physically engage that gear. The window will freeze briefly while waiting — this is normal.

  5. Set the correct keys

    In Key Selection, make sure the Upshift and Downshift keys match what you've configured in the game. See the game-specific tables below for recommended key bindings.

  6. Check Options

    Most DOS racing games need specific option combinations. Refer to the Supported Games table below.

    Typical setup: Reverse is a button ✅ and Nascar mode ✅ for the Papyrus games. Grand Prix 2 needs Memory Write mode.

Step 3 — Run & Play

  1. Click "Run Anyshift"

    The Anyshift window will appear frozen. This is expected — Anyshift is actively running in the background and will respond to your shifter.

  2. Launch your game in DOSBox

    Start DOSBox, navigate to your game folder, and launch the executable. Configure the game's controls to use the same keys you set in Anyshift.

  3. Before the race starts — select first gear

    Engage 1st gear on your shifter before the race begins or before you gain control of the car. This prevents desynchronisation.

  4. Stopping Anyshift

    Press End on your keyboard to stop Anyshift and return to the configuration window.

Game & wheel settings

  • Recommended rotation: 540°

  • Change controls in-game. Default: S upshift, Z downshift
  • Select first gear after the countdown or the game will select reverse on start

General tips for DOSBox games

  • Set combined pedals in your wheel software before starting DOSBox.
  • Configure wheel rotation to match the era of the game (270°–400° for most DOS games).
  • Engage 1st gear before the race starts to avoid sync issues.
  • If gears feel unresponsive, try slightly increasing the Release key delay in Anyshift.
  • Some Papyrus games glitch if you leave a gear selected in the menus — return to neutral position.
