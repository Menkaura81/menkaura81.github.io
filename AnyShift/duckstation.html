How to Use Anyshift with DuckStation

Play PS1 racing games with your H-shifter using the DuckStation emulator and its NeGcon controller support.

📋 On this page

  1. Requirements

  3. Axis & Button Mapping

  5. Initial D Setup

Requirements

  • DuckStation — download from duckstation.org. Any recent version works.
  • A steering wheel with a steering axis (force feedback not supported in DuckStation PS1 emulation)
  • An H-pattern gear shifter recognised by Windows
  • Anyshift installed and working
  • Your PS1 game ROM/image (Gran Turismo, Initial D, etc.)
ℹ Why NeGcon?
The PlayStation 1 did not have an official steering wheel peripheral that PS2 wheels can emulate. DuckStation supports the NeGcon controller — a special analog controller that uses rotation for steering and analog face buttons for throttle/brake. It's the best way to use a steering wheel with PS1 games.

Step 1 — Configure NeGcon in DuckStation

  1. Open DuckStation Controller Settings

    In DuckStation, go to Settings → Controllers.

  2. Select NeGcon for Controller Port 1

    Click on Controller Port 1 in the left sidebar. From the controller type dropdown, select NeGcon.

    ⚠ Do not use Digital Controller or Analog Controller
    Only NeGcon provides proper analog steering input that maps to your wheel's steering axis. Other controller types won't give you proportional steering.

Step 2 — Map Steering Axis & Pedals

With NeGcon selected you'll see the controller binding screen. Map these inputs:

NeGcon Input Type Bind to Notes
Steering / Twist Analog axis Your wheel's steering axis Left = full left lock, Right = full right lock
I button Analog axis Throttle axis (accelerator pedal) I is the main analog button on the NeGcon — used for acceleration
II button Analog axis Brake axis (brake pedal) II is the secondary analog button — used for braking
ℹ Pedal mode
Most wheels that don't natively support NeGcon will need separated pedals mode (each pedal on its own axis). This is opposite to the DOSBox requirement. Check your wheel software — set pedals to Separated if available, and map throttle and brake axes independently in DuckStation.

Recommended wheel rotation: 720° for most PS1 racing games.

Step 3a — Gran Turismo 1 Button Mapping

Gran Turismo 1 uses these controls for manual shifting and reverse:

Game action NeGcon button Bind to key Anyshift default
Upshift D-Pad Up S Upshift key
Downshift D-Pad Down Z Downshift key
Reverse B button C Reverse key
⚠ In-game menus
After mapping D-Pad Up to S and D-Pad Down to Z, these keys will also navigate menus. Use S to go up and Z to go down in all in-game menus.

Anyshift Options for Gran Turismo 1

Detect Neutral❌ Off
Reverse is a button✅ On
Use seventh gear❌ Off
Nascar mode❌ Off

Select first gear on the shifter before going to the track to avoid desynchronisation.

Step 3b — Initial D Button Mapping

Initial D uses shoulder buttons (L / R) for shifting and has no reverse gear:

Game action NeGcon button Bind to key Anyshift default
Downshift L (left shoulder) Z Downshift key
Upshift R (right shoulder) S Upshift key
Reverse — — No reverse in this game

Anyshift Options for Initial D

Detect Neutral❌ Off
Reverse is a button❌ Off
Use seventh gear❌ Off
Nascar mode❌ Off

Step 4 — Configure Anyshift

  1. Launch Anyshift

    Run Anyshift.exe as Administrator if needed.

  2. Load the Gran Turismo 1 preset (if using GT1)

    In Profiles, select Gran_Turismo1_Duckstation from the dropdown and click Load profile.

  3. Select your shifter device

    In Joystick selection, pick your H-shifter. Enable the clutch if you have one.

  4. Verify Key Selection matches DuckStation

    Ensure the Upshift, Downshift, and Reverse keys in Anyshift are identical to what you bound in the DuckStation NeGcon controller config.

    Anyshift fieldKey
    UpshiftS
    DownshiftZ
    ReverseC

  5. Set the correct Options

    For Gran Turismo 1: enable Reverse is a button. For Initial D: leave all options unchecked.

  6. Click "Run Anyshift", then launch the game

    The Anyshift window will freeze — this is normal. Launch DuckStation and start your game. Select manual transmission in the game settings.

    Press End to stop Anyshift.

PS1 Games Compatible with NeGcon

Many PS1 racing games support the NeGcon controller, which means they work with a steering wheel in DuckStation. A comprehensive compatibility list is maintained at:

https://www2.gvsu.edu/brittedg/Negcon.html →

Notable titles known to work:

Game Reverse Upshift button Downshift button
Gran Turismo 1 C (B button) S (D-Pad Up) Z (D-Pad Down)
Gran Turismo 2 C (B button) S (D-Pad Up) Z (D-Pad Down)
Initial D No reverse S (R shoulder) Z (L shoulder)
Ridge Racer Type 4 Uses automatic shifting — test manually
Ace Combat Analog control but different button layout — test manually

Button mappings may vary between game regions and versions. Check the game's controller settings if the defaults don't work.
