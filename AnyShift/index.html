What is Anyshift?

Anyshift is a Windows application written in Python. It reads input from your joystick-based H-pattern gear shifter and translates it so any racing game understands it — whether through simulated keystrokes or direct memory writes.

🎹 Mode 1 — Simulated Keystrokes

Anyshift translates your H-shifter position into sequential up/downshift key presses. For example, shifting from 2nd to 4th automatically fires two upshift keystrokes. Works with any game that has configurable keyboard controls.

Set the same key in Anyshift's Key Selection box and in your game's controller config.

🧠 Mode 2 — Memory Write

Anyshift writes the current gear value directly to the game's memory address. This is the most reliable method — no desynchronisation ever. Requires knowing the memory address for the gear variable (find it with Cheat Engine).

⚠ Warning
Never use Memory Write mode in online games. You may get banned.
💡 Tip
If Anyshift is not working on your system, try running it as Administrator.

Quick Start

  1. Download & extract Anyshift

    Unzip Anyshift to any folder. Run Anyshift.exe (or python src/Main.py for the source version). If it doesn't start, try running as Administrator.

  2. Load a preset for your game

    In the Profiles section, select a preset from the dropdown and click Load profile. Presets are included for GT4, Enthusia, Grand Prix 2, NASCAR Racing and more.

  3. Select your shifter device

    In Joystick selection, pick your H-shifter from the dropdown. Optionally enable and select the clutch device.

  4. Map your shifter buttons

    Click each gear button in the UI, then physically engage that gear on your shifter to record the button number. Repeat for all gears and reverse.

  5. Configure your game & press Run

    Make sure your game uses the same keys as Anyshift's Key Selection box (defaults: S upshift, Z downshift, C reverse). Then click Run Anyshift, launch your game, and enjoy!

    To stop Anyshift press the End key on your keyboard.
