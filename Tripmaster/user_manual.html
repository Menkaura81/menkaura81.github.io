Classic Rally Tripmaster Instrument Description

Classic Rally Tripmaster

Android application — GPS-based distance instrument for rally co-drivers

Purpose

The Classic Rally Tripmaster is a faithful simulation of the classic mechanical Classic Rally Tripmaster used by co-drivers in vintage and historic rally motorsport.

Instead of a mechanical drive cable connected to the gearbox, the app uses the phone's GPS receiver to measure distance with high accuracy. The instrument panel is drawn pixel-for-pixel to replicate the look and feel of the original analogue instrument.

The app is designed for use on the co-driver's side of a rally car — mounted in portrait orientation, readable at a glance, and operable with gloves.

GPS Tracking

The app requests GPS updates at the highest available accuracy. In practice, consumer hardware delivers approximately one fix per second (1 Hz). Readings with an accuracy worse than 20 m are discarded; movements smaller than 2 m are ignored to suppress noise.

GPS distance accumulation is active only when the mode knob is in position + or −. In position 0 the GPS is paused and the counter holds its value.

Operating Instructions

Measuring a Distance

  1. Reset the counter to zero by tapping the Reset Knob (top right).
  2. Set the Mode Knob to position +. The "+" label on the panel illuminates.
  3. Begin driving. The GPS will accumulate distance and advance the counter automatically.
  4. To pause accumulation — at a service park or whilst stationary — set the Mode Knob to 0. The counter holds its value.
  5. Resume by returning the Mode Knob to +.
  6. When not in use, leave the Mode Knob at 0.

Measuring Remaining Distance

  1. Set the counter to the total distance to be covered using the Side Knob or individual digit tap.
  2. Set the Mode Knob to −. The "−" label illuminates.
  3. Begin driving. The counter counts down towards zero.
  4. When the counter reaches 000.00, the set distance has been covered.

Manual Correction

  1. At any time, use the Side Knob to fine-correct the counter against a known distance marker or roadbook reference.
  2. Swipe up to add distance; swipe down to subtract.
  3. Each swipe tick changes the reading by exactly 0.01 km (10 m).
  4. The Mode Knob position does not affect manual correction — the side knob always works.

Setting a Predetermined Distance

  1. With the Mode Knob at 0, use the Side Knob to dial in the starting distance from the roadbook.
  2. Alternatively, tap individual counter digits to set each figure. Tap = +1, long-press = −1.
  3. Set the Mode Knob to + or − as appropriate before the stage starts.

Notes on GPS Accuracy

Consumer GPS hardware typically provides one position fix per second regardless of the update rate requested by the app. For best accuracy:

  • Allow the phone to acquire a strong GPS signal before the stage.
  • Keep the phone with a clear view of the sky. Avoid placing it flat on the dashboard where the roof obstructs satellites.
  • The 20 m accuracy filter rejects poor-quality fixes automatically. In heavy tree cover or tunnels, the counter will pause momentarily.
  • GPS measures the path actually driven, including curves — just like a drive cable on the original instrument.

Permissions Required

Location (fine) Required for GPS distance tracking. Without this permission the counter cannot accumulate distance automatically.
Location (background) Not required. The app only tracks when it is visible on screen.

Manual Adjust

Rally Stage Workflow

  1. Before the stage: reset to zero or set a known starting distance.
  2. At the start line: set Mode Knob to +.
  3. Drive. Read the counter against the roadbook.
  4. Correct with the Side Knob at known distance markers.
  5. At finish or service: set Mode Knob to 0.

Countdown Workflow

  1. Set the stage total distance using the Side Knob.
  2. Set Mode Knob to − at the start.
  3. Drive. The counter counts down to 000.00.
  4. 000.00 = end of the set distance.

The Classic Rally Tripmaster is a simulation instrument for use in historic and vintage rally motorsport. Always observe local traffic regulations. GPS accuracy may vary with satellite availability and atmospheric conditions.
